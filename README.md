# OL-UL_HTML

# Nested List Styling (HTML & CSS Only)

A well-structured demonstration of **deeply nested ordered and unordered lists** using **pure HTML and CSS**.  
This project focuses on **correct HTML semantics**, **modern UI design**, **dark mode support**, and **print-friendly styling** — all **without JavaScript**.

---

##  Features

-  **Valid & semantic HTML structure**
-  **Modern Card UI design**
-  **Automatic Dark Mode** (system preference)
-  **Print-friendly layout**
-  **Smooth hover effects**
-  Supports:
- Ordered Lists (`1, A, I, a, i`)
- Unordered Lists (`disc, circle, square`)
-  **No JavaScript used**
-  Lightweight & fast

---

##  Technologies Used

- **HTML5**
- **CSS3**
  - CSS Variables
  - Media Queries
  - `prefers-color-scheme`
  - Print Media Styles
